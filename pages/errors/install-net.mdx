# You must install or update .NET to run this application

<img src="../assets/error-net.jpg" alt="error-net" className="nx-rounded-lg nx-my-2"/>

Помилка `You must install or update .NET to run this application` виникає через відсутність необхідного
.NET пакету, через що не вдається запустити програму.

Завантажте пакет кнопкою нижче, або при виникненні помилки натисніть `Так` після чого вас спрямує на цеж саме посилання.

<button
    className="nx-mt-4 nx-px-4 nx-py-2 nx-rounded-full nx-border  nx-text-xl"
    onClick={() => window.location.href = `https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-7.0.20-windows-x64-installer?cid=getdotnetcore`}
>
    Завантажити .NET 7.0
</button>

Після встановлення оновлення, повторно запустіть програму IPBanGUI.
