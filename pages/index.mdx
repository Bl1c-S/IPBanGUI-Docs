# Torgsoft IPBanGUI
Ласкаво просимо до керівництва з використання Torgsoft IPBanGUI.

## Що таке Torgsoft IPBanGUI

Torgsoft IPBanGUI — це безкоштовна програма, яка забезпечує додатковий рівень захисту для вашого Windows сервера.
Вона працює на основі [IPBan](https://ipban.com/), інструменту для автоматичного блокування IP-адрес зловмисників.
Ця програма ефективно захищає ваш сервер від брутфорс атак RDP, забезпечуючи стабільну і безпечну роботу системи.
