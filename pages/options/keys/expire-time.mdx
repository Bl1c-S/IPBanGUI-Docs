# ExpireTime
Визначає час, після якого буде [скинуто лічильник](/options/events/types#забуто-невдалий-вхід) невдалих спроб входу для конкретної ІР-адреси.

## Значення
<TimeValue/>

## Опис
<Callout type="info" emoji="ℹ️">
    Якщо після невдалої спроби входу минув час, вказаний у ключі,
    лічильник невдалих спроб для цієї ІР-адреси буде скинуто до `0`.
</Callout>

Таким чином, протягом дня можна багато разів вводити неправильний пароль,
але якщо частота спроб буде меншою за значення ключа `ExpireTime`, програма не буде нас блокувати.

<Callout type="warning" emoji="⚠️">
    Якщо зловмисники підбирають пароль з невеликою частотою спроб,
    надто низьке значення ключа `ExpireTime` може завадити їх блокуванню.
</Callout>
